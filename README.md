### Project Overview

This project examines housing affordability patterns and change over time in the area surrounding the Antioch BART station, using spatial and census-based analysis to evaluate how affordability has evolved in relation to regional access and development. The work focuses on identifying trends, spatial disparities, and policy-relevant insights through geospatial data analysis and visualization.

The complete analysis, results, maps, and narrative discussion are presented exclusively in the ArcGIS StoryMap linked below and should be considered the primary project deliverable.

[Live Project Report (ArcGIS StoryMap)](https://storymaps.arcgis.com/stories/551526ed63854b4b9e1ab8954cc7c246)

This GitHub repository contains the reproducible code environment used to perform the analysis shown in the StoryMap. It is intended to demonstrate technical workflow, analytical methodology, and reproducibility for portfolio and review purposes, rather than to serve as a standalone report.
